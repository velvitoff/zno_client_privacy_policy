Accurately represents your app's identity:

The "Тести ЗНО і НМТ" is a mobile application that provides users with examples of "ЗНО" and "НМТ"(state exams held in Ukraine every year for high-school graduates) exam questions using publicly available exam questions from previous exams.

What will this app do with user data?

The only user data received is: name, email and profile picture. These will be saved and re-displayed to the user. The email will be used to reply to messages sent by the user. The email will be used to identify the user. Some users will be granted access to additional content.

How does this app enhance user functionality?

This app provides users with access to past years task questions from "ЗНО" and "НМТ" exams. It also provides the necessary functionality to let users pass the example tests and check the correct answers.
